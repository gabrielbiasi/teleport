- The `tsh` client tool version >= (=teleport.version=).

  ```code
  $ tsh version
  # Teleport v(=teleport.version=) go(=teleport.golang=)
  ```

<TabItem

- The `tsh` admin tool version >= (=teleport.version=), which you can download
  by visiting the
  [customer portal](https://dashboard.gravitational.com/web/login).

  ```code
  $ tsh version
  # Teleport v(=teleport.version=) go(=teleport.golang=)
  ```

- A host, e.g., an Amazon EC2 instance, where you can install the Teleport Auth
  Service and Proxy Service.

- A registered domain name.

</TabItem>
<TabItem scope={["cloud"]}
  label="Teleport Cloud">

- The `tctl` and `tsh` client tools version >= (=teleport.version=).

  You can download these from [Teleport Cloud Downloads](../../cloud/downloads.mdx).

  ```code
  $ tctl version
  # Teleport v(=teleport.version=) go(=teleport.golang=)

  $ tsh version
  # Teleport v(=teleport.version=) go(=teleport.golang=)
  ```
